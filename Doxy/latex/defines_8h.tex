\doxysubsubsection{\texorpdfstring{KAPI}{KAPI}}
{\footnotesize\ttfamily \#define KAPI}



KAPI (Kaffi API) macro for controlling library symbol visibility. 

This macro ensures that functions and variables are correctly exported or imported depending on whether the library is being built or consumed.

Usage\+:
\begin{DoxyItemize}
\item When building the library (KEXPORT is defined)\+:
\begin{DoxyItemize}
\item Windows (MSVC/\+Clang)\+: expands to {\bfseries{declspec(dllexport)}}
\item {\bfseries{Linux/mac\+OS (GCC/\+Clang)\+: expands to \+\_\+\+\_\+attribute}}((visibility("{}default"{})))
\end{DoxyItemize}
\item When consuming the library (KEXPORT is NOT defined)\+:
\begin{DoxyItemize}
\item Windows (MSVC/\+Clang)\+: expands to \+\_\+\+\_\+declspec(dllimport)
\item Linux/mac\+OS (GCC/\+Clang)\+: expands to nothing (default visibility is sufficient)
\end{DoxyItemize}
\end{DoxyItemize}
